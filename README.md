## Run Locally

Clone the project

```bash
  git clone https://github.com/Sonia-saini/-accidental-visitor-9960.git
```

Go to the project directory

```bash
  -accidental-visitor-9960
```

Install dependencies for frontend as well as backend

```bash
  cd reliance
  npm install
```


Start the localhost server

```bash
  cd reliance
  npm start
```
